## ğŸ› Troubleshooting

**Backend won't start:**
```bash
cd backend
python -m pip install -r requirements.txt --upgrade
```

**Frontend errors:**
```bash

```


**ML models not loading:**
```bash

# Train new models
cd backend
python ml/train.py
```
